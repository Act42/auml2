## Тема: Розпізнавання співу птахів (BirdCLEF 2026)

---

### 1. Аналіз існуючих рішень
Для розв'язання задачі BirdCLEF 2026 було проаналізовано підходи попередніх років (зокрема 2025). Основні тенденції включають:
* Використання **CNN архітектур** (EfficientNet, ResNet) для класифікації спектрограм.
* Використання **Ensemble learning** для підвищення стабільності передбачень.
* Аугментацію даних через змішування шумів (Background Noise).

### 2. Exploratory Data Analysis (EDA)
У ході аналізу даних було виявлено наступне:
* **Загальна кількість видів:** 234 [з файлу `audioml2.ipynb`].
* **Дисбаланс класів:** У наявному наборі `train_soundscapes` реально представлено лише **72 види** з 234 можливих .
* **Характеристики сигналів:** Аудіо записано з частотою дискретизації **32 000 Гц**.
* **Особливості:** Дані містять значну кількість фонових шумів (дощ, вітер), що ускладнює виділення "чистого" сигналу.

### 3. Стратегія валідації
Побудовано надійну систему валідації для уникнення перенавчання:
* **Метод:** Розбиття за назвою файлів (Group Split), щоб сегменти одного запису не потрапляли одночасно в навчання та тест.
* **Пропорція:** 80% (52 файли) — навчання, 20% (14 файлів) — валідація .

### 4. Метрики оцінки
Основним критерієм є **ROC AUC**. Це площа під кривою помилок, яка оцінює якість ранжування передбачень:

$$ROC\ AUC = \int_{0}^{1} TPR(FPR) \, dFPR$$

Додатково пропонується використовувати **F1-score** для оцінки точності розпізнавання конкретних видів при високому дисбалансі класів.

### 5. Опис використаних підходів
Було розроблено 3 моделі на основі класичних ознак (MFCC, Mel-spectrogram, Spectral Centroid, ZCR):
1.  **Random Forest (RF):** Ансамбль з 50 дерев, глибина 10.
2.  **Logistic Regression (LR):** Мультикласова регресія (MultiOutputClassifier).
3.  **MLP Neural Net:** Проста нейронна мережа з одним прихованим шаром (64 нейрони).

Всі моделі працювали на ознаках, стиснутих за допомогою **PCA до 128 компонент**.

### 6. Результати
Результати валідації та тестування:

| Модель | Валідаційний ROC AUC | 
| :--- | :---: |
| **Random Forest** | 0.8175 |
| **Logistic Regression** | **0.8454** |
| **MLP Neural Net** | 0.7810 |

**Найкраща обрана модель:** Logistic Regression .

### 7. Кореляція валідації
Аналіз показав:
* Валідаційний скор (0.84) є оптимістичним, оскільки в тесті Kaggle можуть бути види, не представлені в нашому `train_soundscapes`.
* Позитивна кореляція присутня, проте для покращення LB результату необхідно використовувати повний набір даних `train_audio`.

### 8. Порівняння з попередніми змаганнями
У порівнянні з BirdCLEF 2025, поточне змагання вимагає більшої уваги до обробки слабких сигналів та фільтрації специфічних для регіону шумів.

### Гіпотези та висновки
* Використання класичних ML-моделей дає непоганий базовий рівень (baseline), але для високих позицій необхідні Deep Learning підходи (CNN/Transformers).
* PCA ефективно стискає дані, проте може втрачати тонкі часо-частотні особливості співу птахів.
